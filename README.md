# GrupoC - ChatSockets

## Requisitos
```
> Apache Neatbeans 16
> JDK 17.0.5
```

## Pasos para ejecutar el proyecto
```
> Abrir Apache Neatbeas
> Dar click en el boton "Abrir Proyecto"
> Seleccionar el proyecto "chatSockets-GrupoC", donde lo tengamos descargado
> Una vez abierto el proyecto, en palen izquierdo ir abriendo las cuñas (>) de Hasta llegar a:
  "chatSockets-GrupoC > Source Packages > visual"
> Una vez que podemos ver los archivos dentro del visaul: 
  le damos click derecho al archivos "Frm_Server" y seleccionamos al opción de corrrer archivo (Run File), 
  lo mismo se realiza con el archivo "Frm_ChatCliente"
```
